    Ao desenvolver qualquer pipeline para o Data Lake, **seu time passa a ser o responsável pela sua manutenção**. Em caso de quebra e não reparação, ele será excluído do ambiente.

Você deseja um dataset que não está no Data Lake? Você quer plugar seus Dashboards e Reports numa estrutura mais robusta contra falhas?

Nós do CIT ajudamos seu time a dar os primeiros passos para ganhar mais autonomia dentro do Data Lake. Se seu time sabe Python & Git ou SQL & Git, nós apoiamos no desenvolvimento dos primeiros pipelines para trazer os dados para dentro do Lake ou nos pipelines que montam as tabelas dentro do BigQuery e que alimentarão seus reports, dashboards ou modelos de machine learning.
